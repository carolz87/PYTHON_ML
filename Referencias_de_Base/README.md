## 1. O Início

De começo um clássico:

```
print ("Hello World!")
```

Essa função demonstra uma impressão de texto básica em console, na qual se apresenta aquilo indicado dentro dos parênteses '( )' em console, tendo assim a seguinte apresentação

>Hello World!
